# Mode Selection: Timeline Thinking (Practical)

Use this when planning shots from an idea.

## Core principle

Do not treat video generation modes as random buttons.
Treat them as **control layers** over timeline storytelling.

- `t2v`: lock narrative vibe, keep freedom high
- `i2v`: lock opening composition (best for hard cuts)
- `keyframes`: lock A→B transition goal (best for continuous motion)
- `multiref`: lock identity/style consistency across shots

## Quick decisions

### If you want a clear cut (镜头跳切)
Prefer `i2v` (optionally with `multiref`).

Typical cases:
- wide → close-up
- front → side
- indoor → outdoor

### If you want smooth continuity (连续推进)
Prefer `keyframes` (same subject/space/action line).

Typical cases:
- same character continuous motion
- same scene camera push/pan/track
- seamless transition or controlled morph

### If consistency is critical
Add `multiref` to stabilize character/wardrobe/props/style.

### If exploring concept direction
Use `t2v` first, then refine to `i2v`/`keyframes` for production shots.

## Planning mindset

1. Decide edit grammar first: cut vs continuous.
2. Decide consistency pressure: one-off shot vs same-world sequence.
3. Then choose mode and write prompt.

This avoids over-constraining every shot and keeps the timeline cinematic.
